Steam User and Developer Data Analysis API
Este repositorio contiene una API que proporciona funcionalidades para analizar datos de usuarios y desarrolladoras de juegos en la plataforma Steam. La API ofrece varios endpoints que permiten acceder a información relevante sobre los juegos, usuarios y desarrolladoras en Steam.

https://github.com/Remolino777/steam-game-recommendation-system/assets/39866683/d139f53c-53a4-4956-acaf-2a6692489329

Endpoints Disponibles
1. developer/
Descripción: Este endpoint devuelve una lista que incluye la cantidad de ítems, el año y la cantidad de contenido gratuito de las desarrolladoras de juegos en Steam.

2. userdata/
Descripción: Devuelve una lista que incluye la cantidad de dinero gastado por un usuario, el porcentaje de juegos recomendados y la cantidad total de juegos que posee el usuario.

3. best_developer_year/
Descripción: Devuelve una lista de diccionarios que contienen información sobre las tres desarrolladoras que recibieron la mayor cantidad de críticas positivas en un año específico.

4. UserForGenre/
Descripción: Devuelve el usuario que ha jugado más horas en un género de juegos específico, junto con el número total de horas jugadas por ese usuario en dicho género.

5. developer_reviews_analysis/
Descripción: Devuelve una lista con los valores absolutos de las críticas positivas y negativas recibidas por una desarrolladora de juegos en todos los años disponibles en Steam.

Uso de la API
Para utilizar esta API, puedes enviar solicitudes HTTP a los endpoints mencionados anteriormente y recibirás una respuesta en formato JSON con los datos correspondientes. Asegúrate de incluir los parámetros necesarios en tus solicitudes para obtener la información deseada.
